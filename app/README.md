# Food Hungry App

This is the Expo React Native frontend.

## Important Files

- `App.js`: small entry file required by Expo.
- `src/AppScreen.js`: main Food Hungry screen source code.
- `app.json`: Android/iOS/Web app configuration.
- `package.json`: commands and frontend dependencies.

## Run

```bash
npm install
npm run web
```

For Android phone testing:

```bash
npm start
```

Then scan the QR code with Expo Go.
